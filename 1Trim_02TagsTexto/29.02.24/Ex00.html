Desenvolvimento Web

Tecnologias utilizadas:

Front-End

O desenvolvedor front-end é aquele que consegue receber do analista, um projeto gráfico, como por exemplo um
sistema feito no figma. E a partir dele consegue desenvolver a interface a qual o cliente terá acesso. E para isso, o desenvolvedor utilizara o HTML, CSS e o JavaScript.

Back-End

O desenvolvedor do lado back-end, cuida além do proprio servidor, a interface cliente-servidor. Para executar as ações que não aparecem na tela do cliente. Esse, vai saber as linguagens que permitem essa comunicação, como: JavaScript, Java ou PHP.

FullStack

Esse é o desenvolvedor que tem noção tanto do front-end quanto do back-end. Sendo assim, requisitado pela sua versatilidade me trabalhar tanto em interface-cliente e no servidor. Para isso esse tipo de desenvolvedor tem conhcimento sobre as linguagens acima descritas.
