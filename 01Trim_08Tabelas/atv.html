Como fazer uma tabela?

Exemplo de uma tabela:

Nome Lançamento Nota
A ameaça fantasma 1999 4.2
O ataque dos clones 2002 4.1
A vingança dos Sith 2005 4.5
Tabela 1

Como iniciar uma tabela:

A primeira coisa é colocar a tag <table></table>, para iniciar a tabela.

Para definir as linhas, coloca-se a tag <tr></tr>.

Dentro de cada tag de linha, defini-se as tags para criar as colunas.
Se aquela linha for de título, basta criar cada coluna como se fosse uma lista e usando a tag <th></th>.
E se for de contúdo normal, usa-se <td></td>.

Um exemplo com código:

A tabela 1, foi criada com esse código:

<table style="width: 400px;">
    <tr>
      <th>Nome</th>
      <th>Lançamento</th>
      <th>Nota</th>
    </tr>
    <tr>
      <td>A ameaça fantasma</td>
      <td> 1999 </td>
      <td> 4.2 </td>
    </tr>
    <tr>
      <td>O ataque dos clones</td>
      <td> 2002 </td>
      <td> 4.1 </td>
    </tr>
    <tr>
      <td>A vingança dos Sith</td>
      <td> 2005 </td>
      <td> 4.5 </td>
    </tr>
</table>
