Tag <del>

Serve para mostrar que um testo foi removido.

Ex:

  • Texto riscado.
  • ToDo concluido!
  • O texto mudou.

Tag <ins>

Serve pra resaltar um texto. Como os textos inseridos.

Ex:

  • Este texto tem uma linha embaixo!
  • Este texto está resaltado!
  • Pode indicar varias coisas.

Tag <sub>

Serve para deixar um texto menor, e em uma posição abaixo dos demais.

Ex:

  • Pode ser usado para equações: A x + 1 = f(A x)
  • E para formulas químicas: H2O.
  • O texto fica menor. Tipo esse!

Tag <sup>

É um texto menor, assim como o sub. Porém na parte superior da linha.

Ex:

  • Pode ser usado como expoente: ax · ay = ax+y
  • Pode ser usado em rodapés.
  • Texto em fonte reduzida.

Tag <blackquote>

Utilizado para citações. Cria um espaçamento extra com a lateral.

Ex:

"Tudo o que temos de decidir é o que fazer com o tempo que nos é dado."
Gandalf em "A sociedade do anel" escrito por J. R. R. Tolkien.
"Só se vê bem com o coração, o essencial é invisível aos olhos."
Antoine de Saint-Exupéry em "O pequeno príncipe".
"Tentar não. Faça ou não faça. Tentativa não há."
Yoda em "Star Wars: Episódio V".

Tag <q>

É como se fosse um blockquote, porém para citações curtas.

Ex:

  • Ele adiciona as aspas.
  • Não cria espaçamento extra.
  • Não quebra a linha.

Tag <abbr>

Serve para criar abreviações.

Ex:

  • Ele está estudando para a OBMEP
  • Uma das maiores ligas de basquete é a NBA.
  • A NBA fica nos EUA.

Tag <address>

Serve para definir informações de conatto com o proprietario ou autor. Ele gera quebra de linha.

Rua Professora Dionice Taques
399
Jardim Bela Vista
Imbituva, Paraná, Brasil
