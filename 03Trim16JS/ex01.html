São três tiois de caixas de diálogos.

São três os tipos de caixa de diálogo:

  • alert: exibe uma mensagem de alerta para o usuário
  • confirm: exibe uma mensagem de de corfirmação em que o usuário pode escolher entre ok e cancelar.
  • prompt: pode entrar com uma caixa de texto.

É importante destacar que o Hava script pode ser incorporado ao documenro HTML pela tag script. O script js pode ser interno, ou seja, dentro do documenro HTML ou externo js.
